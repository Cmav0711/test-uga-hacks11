# Symbol Templates

This directory contains template images for symbol recognition.

## Usage
Place symbol template images in this directory. Each image should:
- Be a PNG file
- Contain a single symbol/character
- Have a clear, high-contrast representation of the symbol
- Be named descriptively (e.g., "A.png", "star.png", "circle.png")

The symbol detection system will compare captured images against these templates to identify matching symbols.

## Supported Formats
- PNG (recommended)
- JPG/JPEG

## Example Structure
```
symbols/
  ├── A.png
  ├── B.png
  ├── circle.png
  ├── star.png
  └── README.md
```
